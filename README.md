# Asset Management System

A modern **Enterprise Asset Management System** developed using **Laravel** to efficiently manage organizational assets, assignments, employee records, and backend operations. The application follows the MVC architecture and provides a scalable solution for tracking and managing company assets.

---

## 🚀 Overview

The Asset Management System helps organizations maintain complete control over their assets by providing centralized management, assignment tracking, authentication, and backend integrations. It is designed to improve asset visibility, streamline operations, and reduce manual management.

## ⚙️ Installation

```bash
git clone https://github.com/ahsanali35103/asset-management-system.git

cd asset-management-system

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
